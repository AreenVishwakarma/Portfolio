PROJECTS

To-Do List

This To-Do List web application is built using JSP, JQuery,Servlets, and MySQL. It allows users to add tasks through a simple interface, which are then stored in a MySQL database. A servlet handles the task submission and retrieval, ensuring dynamic interaction between the frontend and backend. The tasks are displayed on the main page, allowing users to keep track of their to-dos. The application offers a clean and functional way to manage daily tasks with persistent storage. It's a great example of basic CRUD operations using Java web technologies.

Vote Page

This Voting System web application is developed using JSP, Servlets, and MySQL. Users must first register and log in to access the voting page. Once logged in, they can view a list of candidates and cast their vote. After voting, the system records their choice in the database and automatically logs them out to prevent multiple votes. It ensures one-time voting per user and maintains session control for secure access. The application demonstrates user authentication, session handling, and database interaction in a simple and effective way.

Result

This Student Result Page web application is built using JSP, Servlets, and MySQL. Students can log in using their credentials to view their academic results. Once logged in, the system fetches the student's marks and details from the database and displays them in a structured format. The application ensures secure access through session handling and provides accurate, real-time result viewing. It's a simple and efficient system for managing and displaying student academic performance.

E-Commerce Login

This E-commerce website is developed using JSP, Servlets, and MySQL to provide a dynamic online shopping experience. Users can browse products, view details, add items to their cart, and place orders. The site supports user registration and login, with session management to track user activity and cart items. Product data and order history are stored in a MySQL database for efficient management. The application demonstrates key features of an online store with a user-friendly interface and secure transaction flow.

Catalog

This E-commerce Catalog web application is built using JSP, Servlets, and MySQL to showcase products in an organized and user-friendly manner. Users can browse through various categories, view detailed information about each product, and explore available items without needing to log in. The catalog pulls product data dynamically from a MySQL database, ensuring up-to-date listings. It provides a clean interface for displaying images, prices, and descriptions, serving as the core browsing experience of an online store.

Quiz

The Quiz Page is a secure and interactive platform built for educational assessments. Students are required to log in to access and attempt quizzes designed to test their knowledge. Admin users have the authority to manage the system by adding or removing students and creating new quizzes with ease. The interface is clean, intuitive, and optimized for a seamless user experience. This system ensures efficient quiz management and fosters a smooth learning environment.
