Curd Registration

Add Employee

This Curd Registration Application is developed using JSP, Servlets, and MySQL. Users must first register and log in to access the employee management tool. Once logged in, they can upload employee documents for processing. The application supports various document formats, allowing users to choose their desired output format. It ensures secure file handling through user authentication and session management. The application demonstrates key concepts of web development, including file uploads, real-time processing, and database integration.
